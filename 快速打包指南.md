# Windows 打包快速指南

## 最快的方式（3步）

### 1️⃣ 准备环境（第一次需要）

在 Windows 命令行（cmd）中运行：

```cmd
pip install PyInstaller vosk sounddevice
```

### 2️⃣ 执行打包

方式 A（推荐）- 直接双击：
```
双击 build.bat 文件
```

方式 B - 命令行运行：
```cmd
python simple_build.py
```

### 3️⃣ 获取 exe

打包完成后，进入 `dist` 文件夹，找到 `语音转文字.exe`

---

## 部署方式

### 📦 便携版（推荐）

```
步骤：
1. 复制 vosk-model-small-cn-0.22 文件夹到 dist 文件夹
2. 复制 corrections.json 到 dist 文件夹
3. 将整个 dist 文件夹压缩为 zip
4. 重命名为 语音转文字-v1.0.zip
```

**用户使用**：
- 下载 zip 文件
- 解压即可运行
- 无需安装 Python

### 🌐 网盘分享

支持的平台：
- 微博云盘
- 百度网盘
- OneDrive
- Google Drive
- GitHub Releases

### 🔧 单独 exe（轻量级）

如果只分享 exe，用户需要：
1. 安装 Python
2. 安装依赖：`pip install vosk sounddevice`
3. 下载 `vosk-model-small-cn-0.22` 模型

## 故障排除

### exe 无法运行

```cmd
# 进入 dist 文件夹后尝试
语音转文字.exe --cli
# 查看命令行输出的错误信息
```

### 缺少模型文件

确保 `vosk-model-small-cn-0.22` 存在于：
- 方式1: `dist` 文件夹同级
- 方式2: `dist` 文件夹内部

### Python 相关错误

重新安装依赖：
```cmd
pip install --upgrade vosk sounddevice
```

## 创建卸载脚本（可选）

创建文件 `uninstall.bat`：

```batch
@echo off
REM 卸载脚本
echo 确认删除语音转文字及所有文件？
pause
rmdir /s /q vosk-model-small-cn-0.22
del speech_to_text_unified.exe
del corrections.json
echo 已删除
pause
```

## 版本管理

### 更新程序

当源代码更新时：
1. 替换 `speech_to_text_unified.py`
2. 运行 `python simple_build.py`
3. 新 exe 自动在 `dist` 中生成

### 发布新版本

```
版本号命名：v1.0, v1.1, v2.0 等
文件名：语音转文字-v1.0.zip
```
